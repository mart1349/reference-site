Naming Coventions

Three rules to follow when naming files for the web

  1. All lowercase

    2. You can only name them with lowercase letters because operating systems treat capitalization differently.

  2. No Spaces

    3. URLs can't have spaces in them.

  3. Only letters, numbers, and dashes

Commit Messages

  1. Minimun 3 words & 10 characters
  2. Proper spelling & grammar
  3. Capital starting letter, no period
  4. Must start with an imperative verb: Create, Fix, Add, Solve, etc.
  5. Pretend every commit starts with the phrase: “This commit will…”
